# Asifam - Asistente Familiar Inteligente

Asifam es un asistente de voz y recordatorios diseñado para tablets Android en el hogar. Permite programar avisos familiares que se ejecutan automáticamente con voz (TTS) y una interfaz visual atractiva.

## Características
- **Recordatorios de Voz**: Notificaciones automáticas por voz para tareas diarias.
- **Interfaz Proactiva**: Una "cara" animada que reacciona cuando el asistente habla.
- **Persistencia Local**: Uso de Isar Database para almacenamiento rápido y seguro.
- **Siempre Activo**: Configurado para mantener la pantalla encendida 24/7.
